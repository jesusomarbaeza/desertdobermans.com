  DESERT DOBERMANS — Litters Page Settings
============================================================

  HOW TO EDIT THIS FILE:
  - Each litter starts with:  LITTER: Display Name Here
  - Below it, add:            FOLDER: FolderName
    (this is the folder name inside Settings/Litters/)
  - Lines starting with # are comments (ignored).
  - Do NOT change the labels themselves (the part before the colon).

  HOW TO ADD PUPPY PHOTOS:
  - Drop image files (.png, .jpg, .jpeg, .webp, etc.) into the
    matching folder inside Settings/Litters/.
  - After adding/removing photos, update Settings/Litters/images.json
    for that litter folder (add/remove the exact filename).
  - Keep filenames exactly as they appear in the folder
    (including uppercase/lowercase).
  - To remove a puppy, just delete its image file.
  - To add a new litter, create a new folder and add a
    [LITTER: / FOLDER:] pair below, then add that folder to
    Settings/Litters/images.json.
