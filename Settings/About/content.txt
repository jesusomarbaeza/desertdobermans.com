  HOW TO EDIT THIS FILE:
  - Each dog starts with:       SECTION: Dog Name Here
  - Each card needs three lines:
        ORG:   Organization name (Embark, OFA, AKC, BBB).
          Must match an image in Settings/About/Icons/.
      LABEL: Button text (e.g. DNA RESULTS, CERTIFICATIONS).
      LINK:  URL or local file path.
  - URLs open in a new tab.
    - File paths (e.g. Settings/About/Certs/…) open the PDF.
  - To add a new card, copy an existing ORG/LABEL/LINK block.
  - To add a new dog, add a new SECTION: line above the cards.
  - Lines starting with # are comments (ignored).
  - Do NOT change the labels themselves (the part before the colon).
